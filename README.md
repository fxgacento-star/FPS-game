# Battle Royale FPS Game

A PUBG-like Battle Royale FPS game built with **Unity** and **Netcode for GameObjects**.

## Project Overview

This project aims to create a multiplayer Battle Royale game featuring:
- Multiple players dropping onto a large map
- Shrinking safe zone mechanic
- Looting system with weapons and items
- Real-time player-versus-player combat
- Inventory management
- Networking and multiplayer synchronization

## Tech Stack

- **Engine**: Unity (Latest Version)
- **Language**: C#
- **Networking**: Netcode for GameObjects
- **Version Control**: Git & GitHub

## Project Structure

```
Assets/
├── Scripts/              # All C# game logic
│   ├── Player/          # Player controller, movement, health
│   ├── Networking/      # Multiplayer sync, spawning
│   ├── GameManager/     # Game state, zone, logic
│   ├── UI/              # UI systems (HUD, menus, etc.)
│   ├── Weapons/         # Weapon systems and mechanics
│   ├── Items/           # Looting and inventory
│   └── Utilities/       # Helper functions and managers
├── Scenes/              # Game scenes
├── Prefabs/             # Reusable prefabs
├── UI/                  # UI canvases and layouts
├── Audio/               # Sound effects and music
├── Models/              # 3D models and textures
└── Resources/           # Configs and scriptable objects
```

## Getting Started

1. Clone this repository
2. Open with Unity (Latest Version)
3. Install Netcode for GameObjects via Package Manager
4. Start developing!
